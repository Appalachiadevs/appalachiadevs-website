# Appalachia Devs — Homepage

Professional, simple marketing website for Appalachia Devs (Miami, Florida).

## Stack
- React + TypeScript
- Vite

## Getting started
1. Install Node.js (LTS recommended).
2. Install dependencies:
   - `npm install`
3. Start dev server:
   - `npm run dev`
4. Build for production:
   - `npm run build`
5. Preview production build:
   - `npm run preview`

## Update company info
Edit `src/siteConfig.ts` to update email, phone, and scheduling link.

## Logo
- Current logo is a placeholder at `public/logo.svg`.
- Replace it with your real logo by overwriting `public/logo.svg`, or add `public/logo.png` and update `src/siteConfig.ts`.
